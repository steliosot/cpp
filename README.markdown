# C++ Foundations Lab

This repository contains teaching material for a beginner C++ lab on decision-making programs and data processing.

The lab introduces:

- basic C++ syntax
- input and output
- variables
- conditionals
- loops
- a simple real-world scenario (fake news detection)

The material follows the lecture slides used in class deck.

### Files

- You can download slides used in the class (PDF). The slides introduce C++ foundations, decision logic, and loops for simple data-driven programs deck.
- Installation Guide on jow to install C++ on Mac, Windows, and Linux. It also includes an online compiler option for quick testing installation.

* Main student lab sheet. It is designed for beginners with no programming background exercises.

---

### How to use this repository

1. Read the installation guide → [Intallation Guide](installation.md)
2. Download or open the exercises → [Exercises](exercises.md)
3. Follow the tutorial step by step
4. Complete the exercises
5. Run all programs locally

---

### Learning outcomes

After completing this lab, students should be able to:

- compile and run a C++ program
- read input from users
- use variables
- write `if` statements
- write `for` loops
- build a simple decision program
